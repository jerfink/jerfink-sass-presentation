SASS

"Syntactically Awesome Style Sheets"


							.container {
							  display: block;
							  float: left;
							  width: 25%;
							  background-color: #f5f5f5;

							  .container_inner {
							    padding: 20px;
							    border: 1px solid white;
							    transition: all 200ms ease 0s;
							  }
							}

Compiles this CSS:


.container{display:block;float:left;width:25%;background-color:#f5f5f5;}.container .container_inner{padding:20px;border:1px solid white;-ms-transition:all 200ms ease 0s;-moz-transition:all 200ms ease 0s;-webkit-transition: all 200ms ease 0s;transition:all 200ms ease 0s;}
